# Project Title: Punjab Click TV Backend

## Description
This project is the backend for the Punjab Click TV application, built using Express and TypeScript. It provides a RESTful API for the frontend to interact with.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd pct-backend
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
To start the server, run:
```
npm run start
```
Make sure to have the necessary environment variables set up.
